# Group 1 Reto CRUD Avanzado

Repository Link: [Reto CRUD Avanzado](https://github.com/School-Group-1/Reto_CRUD_Avanzado_DA.git)
Database Name: CRUD

## Project Setup

1. Clone the repository (git clone https://github.com/School-Group-1/Reto_CRUD_Avanzado_DA.git)
2. In netbeans, open the clone project located in the project directory
3. Configure the TestFX and JasperReports libraries in the project properties
4. Inside the resources package, configure the database
5. Create the blank database using the sql provided in the project directory (Hibernate loads the data into the datbase and creates the tables)
6. Run the project

## Changes from the original project

We decided to change the CSS of the whole application but still maintain all of the same logic from the original application. The only thing that changed was the design of the windows, the rest of the logic stayed the same but migrated to hibernate
